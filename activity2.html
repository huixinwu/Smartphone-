Activity 2: links and navigation tabs

Description

For activity 2, students must create a application of something, for example, an app about a city, an animal, a product, etc. The app has to have the basic elements learned in class such as the use of lists, images, and links.

The requirements are:

  • title using a heading.
  • introduction paragraph with a hyperlink in one of the text.
  • introduction image links to a website.
  • two images in a row. Both images have to link to a website.
  • internal hyperlinks in a list (ordered or numbered list).
  • three social media icons link to its respective website.
  • add a footer at the end with the following information: activity 2 @ by _______
