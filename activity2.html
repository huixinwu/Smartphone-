Activity 2: links and navigation tabs

Description

For activity 2, students must create an application about something, for example, an app about a city, an animal, a product, etc. The app must have the basic elements learned in class such as the use of lists, images, and links, plus the elements learned in the previous class (p, em, hr, and at least two different headings)

The requirements are:

Activity 2 requirements and grading
Elements Points
Title using a heading 0.5
Introduction paragraph with a hyperlink in one of the word 1
Introduction image with an external link to a website 1
Two images in a row. Both images must have a link to a website 1.5
Internal links that link to at least two different sessions within the app. 1.2
Use one list (ordered or unordered list) 0.8
