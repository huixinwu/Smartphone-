Project 1: informative mobile web app

Download Project 1 Instructions

Description


Project 1 consists of creating a one-page informative app. The topic is free to choose, you can talk about places, concepts, nature, beauty, cars, etc..., where you want to inform people about it. This information must have four sections: introduction, information 1, information 2, and contact. For example, if you picked Japan as your topic, then you want to provide the following information to the user:

  1. Introduction: where is the place and why it is a famous place.
  2. Information 1: Activities to do in that place.
  3. Information 2: Important things or locations of that place.
  4. Contact: contact information

Remember that you are creating a one page web app using the HTML skills learned in Week/Day 1, 2, 3, and 4 of class. The HMTL skills that must be in your websites are:
  • Writing links: linking to other sites, to an email address, and to a specific part of the same page
  • Text characteristics, text-size, text alignment, text shadow, text border, margin and padding
  • Design layout in CSS using flex element (properties), and box-sizing. You can also use float properties instead of flex properties.
  • CSS styling images: rounded, circled, border, and shadow images.
  • Navigation Bars (fixed navigation bar).
  • Create contact and media icons.
