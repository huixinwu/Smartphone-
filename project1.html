Project 1: informative mobile web app

Description


Project 1 consists in creating an informative app. The information of the app is about a place (city, country, or state) where you want to inform people about it. In your web app, you should present four informations about the place such as:

  1. The location of the place and why it is a famous place.
  2. Activities to do in that place.
  3. Important things or locations of that place.
  4. Show photos of that place.

Remember that you are creating a one page web app using the HTML skills learned in Week/Day 1, 2, 3, and 4 of class. The HMTL skills that must be in your websites are:
  • Manipulation of basic HTML tags: paragraph, headings, bold, italic, line breaks, horizontal rules, strikethrough, abbreviations.
  • Create ordered or unordered list and definition list.
  • Use and manipulate images and logos using figure and figcaption, and some imagesâ€™ attributes such as title, alt, width, height, and align.
  • Create links to other sites and how to open links in a new window; link to an email address; links to a specific part of the same page; links from an image; and links a webpage with the same site.
  • Create navigation hyperlinks using unordered list and nav tag.
