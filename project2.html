Project 2: sales mobile app

Download Project 2 Instructions

Description


Project 2 consists of creating a sales page. The purpose of project 2 is to use the knowledge of colors, image manipulation, animation, modal window, and slideshow to create a simple sales app.


Procedure

  1. Prepare your development environment for project 2. In the text-editor, make sure you have an index.html and a style.css file. Also create a new folder for images.
  2. Create a header, with a logo and title, to the top of the app.
  3. Create one slideshow with five slides, delay between slides, and backward to the beginning of the slides. The slideshow could be only text, images, or text and images.
  4. One introduction image with filter and a text message in front of the image. The text message should have border, border-radius, title, text with different colors, and two internal hyperlinks link to two different sections in the app.
  5. Present at least four products. Each product should have an image with a text message in front of it, information paragraph, price, and a hyperlink. The hyperlink will open a modal window.
  6. Each modal window should have a box-shadow, title, close symbol “X”, an image, a description paragraph, a footer section with ADD TO CART link. When the modal window is closed, it should go back to its respective product. For example, if you open the modal window for product 3, when you close the modal window of product 3, the app should go to product 3. The other three product should have the same style as the first product.
  7. The last section of project 2 is Contact Us. The contact us section should have an email icon linked to an email, and a phone icon with a phone number. Also, within the Contact Us section, it should have three social media icons. Each icon should be linked to its respective URL in a new window.
  8. Add a footer to the end of the app. The footer of the app should have the following credential: Designed by________ (student’s name) @ 2023 The information and material used is for educational purpose and not commercial purpose
